| Module | Logic Pattern | User Experience |

| **Global Lock (GSB)** | `[Sec_Proc]` touches `[Path]` -> `< 0.5s` later `[AT_Proc]` fails `[Path]` with `SHARING_VIOLATION`. | System 'hiccups' or AT speech skips. Detects 'Security Fratricide'. |
